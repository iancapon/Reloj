Reloj Del sistema solar siguiendo las leyes de kepler...

"Barra espaciadora" -> pausar / empezar
"0 1 2 3 4 5 6 7 8" -> centrar en cuerpos celestes
"scroll"            -> acercar / alejar
"R"                 -> reiniciar el trazo de las órbitas
"+"                 -> acelerar simulación (dias por segundo)
"-"                 -> realentizar simulación

Para correrlo:
-> npm install
-> npm start
-> abrir el link
